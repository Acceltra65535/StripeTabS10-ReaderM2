🏗️ 项目架构
───────────────────────────────────────────────────────────────────────────

核心组件：
┌─────────────────────────────────────────────┐
│           TerminalViewModel                  │  - 业务逻辑编排
│  ┌─────────────┬──────────┬─────────────┐  │
│  │ Backend     │ Receipt  │ USB Monitor │  │  - 支撑服务
│  │ Client      │ Manager  │             │  │
│  └─────────────┴──────────┴─────────────┘  │
└─────────────────────────────────────────────┘
          ↓
┌─────────────────────────────────────────────┐
│           TerminalFragment                   │  - UI界面
│  (状态观察 → 按钮响应 → 结果显示)           │
└─────────────────────────────────────────────┘
          ↓
┌─────────────────────────────────────────────┐
│         Stripe Terminal SDK                  │  - 支付处理
│  (Device Discovery → Payment → Confirmation)│
└─────────────────────────────────────────────┘

完整支付流程：
1. 应用启动 → 初始化Terminal SDK
2. 用户输入 → 金额、货币、设置
3. USB发现 → 发现Reader M2
4. 连接 → 建立与Reader的连接
5. 支付意图 → 从后端获取client secret
6. 收集 → 收集支付方法
7. 确认 → 向Stripe确认支付
8. 收据 → 生成并保存收据
9. 结果 → 显示成功/失败信息

🚀 快速开始
───────────────────────────────────────────────────────────────────────────

1. 环境准备
   ./gradlew clean build

2. 安装应用
   adb install app/build/outputs/apk/debug/app-debug.apk

3. 配置终端
   打开应用 → Settings → 填写Stripe Location ID和后端URL

4. 连接Reader
   点击 "Discover USB reader" → 选择 → 连接

5. 进行支付
   输入金额 → 点击 "Collect & charge" → 完成支付
